# RADIO IVANPLAY ESPAÑA — sitio estático
Estructura:
/
  index.html
  .nojekyll
  /music
    playlist.json
    (pon aquí tus MP3/WAV)

Pasos:
1) Copia tus canciones dentro de /music (ej. track01.mp3, track02.mp3, ...).
2) Si cambias nombres, edita /music/playlist.json para que los apunte correctamente.
3) Sube todo el contenido de esta carpeta al repositorio: ivanplayradio.github.io
4) Abre: https://ivanplayradio.github.io

NOTAS:
- El reproductor usa sólo AutoDJ (sin stream).
- Shuffle activado.
- En móviles debes pulsar 'Reproducir' por políticas de autoplay.
